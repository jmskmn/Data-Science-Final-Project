# Data Science Final Project

The goal of this project is to evaluate potential causes of hit and runs based on car crash data from Chicago by running logistic regression and picking out the most impactful parameters that the model uses.

To see my methods and results, click on the "Data_Science_Project (1).ipynb" file and scroll through. Since the raw data file is too large to upload here, you can download it from [kaggle](https://www.kaggle.com/datasets/nathaniellybrand/chicago-car-crash-dataset/data) if you wish.
